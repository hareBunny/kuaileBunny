# 🚨 白屏问题 - 3分钟快速解决

## 问题

访问 https://kuaile8.pages.dev 显示白屏，控制台错误：
```
main.tsx:1 Failed to load module script
```

---

## 原因

Cloudflare Pages 的 **Build output directory** 配置错误。

---

## 解决方案（二选一）

### 🥇 方案1: 修改Dashboard配置（推荐）

1. 访问 https://dash.cloudflare.com
2. Workers & Pages → kuaile8 → Settings → Builds & deployments
3. 点击 **Edit configuration**
4. 修改 **Build output directory** 为: `h5/dist`
5. 点击 **Save**
6. 回到 Deployments → 点击最新部署 → **Retry deployment**
7. 勾选 **Clear build cache** → 点击 **Retry**
8. 等待2-3分钟
9. 访问 https://kuaile8.pages.dev 验证

---

### 🥈 方案2: 使用CLI部署（快速）

```bash
# Windows用户直接运行
deploy-pages-cli.bat

# 或手动执行
cd h5
npm run build
npx wrangler pages deploy dist --project-name=kuaile8
```

**首次使用需要**:
```bash
npm install -g wrangler
wrangler login
```
