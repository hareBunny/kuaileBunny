## 🐛 常见问题

### 问题 1: 页面空白

**解决方案**:
1. 检查浏览器控制台是否有错误
2. 确认前端服务是否运行
3. 刷新页面（Ctrl + F5）

### 问题 2: API 请求失败

**解决方案**:
1. 检查后端是否运行：http://localhost:8787
2. 查看 Network 标签的错误信息
3. 检查 vite.config.ts 的 proxy 配置

### 问题 3: 登录后刷新丢失状态

**解决方案**:
1. 检查 localStorage 中是否有 user-storage
2. 清除浏览器缓存重试
3. 重新登录

### 问题 4: 数据库没有数据

**解决方案**:
```bash
cd backend-cloudflare
wrangler d1 execute kuaile8 --file=./seed-test.sql
```
