## 🚀 新增功能

### 1. 多数据源支持

现在支持4种数据源：

1. **数字星空API**（推荐）
   - 完全免费
   - 接口稳定
   - 支持历史数据
   - 注册地址：https://szxk365.com

2. **极速数据API**
   - 知名平台
   - 接口规范
   - 注册地址：https://www.jisuapi.com

3. **接口盒子API**
   - 原有方案
   - 可能需要付费
   - 注册地址：https://www.apihz.cn

4. **官网爬取**
   - 备用方案
   - 不稳定（有反爬）

### 2. API测试工具

一键测试所有API是否可用：

```bash
cd backend-cloudflare/crawler
npm run test:api
```

### 3. 新增npm命令

```bash
# 数字星空API
npm run crawl:szxk              # 获取最新数据
npm run crawl:szxk:history      # 获取历史100条数据

# 极速数据API
npm run crawl:jisu              # 获取最新数据

# API测试
npm run test:api                # 测试所有API

# 原有命令
npm run crawl                   # 接口盒子API
npm run crawl:official          # 官网爬取
npm run schedule                # 启动定时任务
npm run pm2:start               # PM2启动
```

### 4. 灵活的爬虫选择

通过环境变量选择使用哪个爬虫：

```bash
# 使用数字星空API（默认）
export CRAWLER_SCRIPT=szxk-crawler.js

# 使用极速数据API
export CRAWLER_SCRIPT=jisu-crawler.js

# 使用接口盒子API
export CRAWLER_SCRIPT=simple-crawler.js

# 启动定时任务
npm run pm2:start
```

### 5. 历史数据获取

数字星空API支持批量获取历史数据：

```bash
# 获取历史100条
npm run crawl:szxk:history

# 获取指定数量
node szxk-crawler.js --history 500
```

## 📖 使用指南

### 快速开始（5分钟）

1. **安装依赖**
   ```bash
   cd backend-cloudflare/crawler
   npm install
   ```

2. **测试API**
   ```bash
   npm run test:api
   ```

3. **注册获取API KEY**
   - 访问 https://szxk365.com
   - 注册账号
   - 获取API KEY

4. **配置环境变量**
   ```bash
   # Windows CMD
   set SZXK_API_KEY=你的API_KEY

   # Windows PowerShell
   $env:SZXK_API_KEY="你的API_KEY"

   # Linux/Mac
   export SZXK_API_KEY=你的API_KEY
   ```

5. **测试爬取**
   ```bash
   npm run crawl:szxk
   ```

6. **启动定时任务**
   ```bash
   npm run pm2:start
   ```

## 🐛 常见问题

### Q1: 为什么要更新？

A: 原有的接口盒子API需要付费，新增的数字星空API完全免费且功能更完整。

### Q2: 原有的爬虫还能用吗？

A: 可以，所有原有文件都保留了，你可以继续使用接口盒子API。

### Q3: 如何选择API？

A: 推荐使用数字星空API（免费、稳定）。运行 `npm run test:api` 可以测试所有API。

### Q4: 需要重新部署吗？

A: 不需要，只需要配置新的API KEY并重启定时任务即可。

### Q5: 数据会丢失吗？

A: 不会，所有数据都保存在Cloudflare D1数据库中，更换API不影响已有数据。
