## 🚀 下一步操作

### 方案A: 继续使用演示数据

```bash
# 生成更多测试数据

```

### 方案B: 接入真实API

1. **注册API KEY**
   - 推荐：https://szxk365.com （免费）
   - 备选：https://www.jisuapi.com （免费）

2. **配置环境变量**
   ```bash
   # Windows CMD
   set SZXK_API_KEY=你的API_KEY

   # Windows PowerShell
   $env:SZXK_API_KEY="你的API_KEY"
   ```

3. **测试真实爬虫**
   ```bash
   npm run test:api
   npm run crawl:szxk
   ```

4. **启动定时任务**
   ```bash
   npm run pm2:start
   ```
