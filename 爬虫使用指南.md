## 三种运行方式

### 方式1: 手动执行（测试用）

```bash
cd backend-cloudflare/crawler
npm run crawl
```

**适用场景**: 测试、调试、手动补数据

### 方式2: 定时任务（生产环境）

```bash
cd backend-cloudflare/crawler
npm install
npm run schedule
```

**特点**:
- 每小时自动执行
- 后台运行
- 自动重试

**使用 PM2 管理**:

```bash
# 安装 PM2
npm install -g pm2

# 启动定时任务
pm2 start scheduler.js --name kuaile8-crawler

# 查看状态
pm2 status

# 查看日志
pm2 logs kuaile8-crawler

# 停止
pm2 stop kuaile8-crawler

# 重启
pm2 restart kuaile8-crawler

# 开机自启
pm2 startup
pm2 save
```

### 方式3: Cloudflare Workers 定时任务（推荐）

已经在 `backend-cloudflare/src/cron.ts` 中实现，无需额外配置。

Cloudflare 会自动每5分钟执行一次，从API获取最新数据并保存到D1数据库。

## 数据验证

### 验证规则

1. **期号格式**: 11位数字 (YYYYMMDDXXX)
2. **号码数量**: 20个
3. **号码范围**: 1-80
4. **号码唯一**: 不重复
5. **总和范围**: 210-1540

### 验证代码

```javascript
function validateData(data) {
  // 验证期号
  if (!/^\d{11}$/.test(data.draw_no)) {
    return false
  }
  
  // 验证号码
  if (data.numbers.length !== 20) {
    return false
  }
  
  for (const num of data.numbers) {
    if (num < 1 || num > 80) {
      return false
    }
  }
  
  // 验证唯一性
  const unique = new Set(data.numbers)
  if (unique.size !== 20) {
    return false
  }
  
  // 验证总和
  if (data.sum < 210 || data.sum > 1540) {
    return false
  }
  
  return true
}
```

## 常见问题

### Q1: 爬虫无法获取数据？

**解决方案**:
1. 检查网络连接
2. 检查API是否可用
3. 查看错误日志
4. 尝试手动访问URL

### Q2: 数据重复？

**解决方案**:
使用 `INSERT OR IGNORE` 自动去重，无需担心。

### Q3: 数据不完整？

**解决方案**:
1. 检查号码数量
2. 验证数据格式
3. 手动补充缺失数据

### Q4: 定时任务不执行？

**解决方案**:
1. 检查 PM2 状态：`pm2 status`
2. 查看日志：`pm2 logs`
3. 重启任务：`pm2 restart kuaile8-crawler`

## 性能优化

### 1. 并发控制

避免同时发起过多请求：

```javascript
// 添加延迟
await sleep(1000)
```

### 2. 缓存机制

缓存已爬取的数据，避免重复爬取：

```javascript
// 检查是否已存在
const exists = await db.prepare(
  'SELECT 1 FROM lottery_results WHERE draw_no = ?'
).get(drawNo)

if (exists) {
  console.log('数据已存在，跳过')
  return
}
```

### 3. 错误重试

失败后自动重试：

```javascript
async function fetchWithRetry(url, maxRetries = 3) {
  for (let i = 0; i < maxRetries; i++) {
    try {
      return await axios.get(url)
    } catch (error) {
      if (i === maxRetries - 1) throw error
      await sleep(1000 * (i + 1))
    }
  }
}
```
