# 后端实现文档

## 项目概述

快乐8百宝箱后端服务，基于 Next.js 14 + MongoDB 构建的 RESTful API。

## 技术栈

- **框架**: Next.js 14 (App Router)
- **数据库**: MongoDB Atlas
- **认证**: JWT (jsonwebtoken)
- **密码加密**: bcryptjs
- **语言**: TypeScript

## 数据库设计

### 1. users 集合

用户信息表

```typescript
{
  _id: ObjectId,
  phone: string,                    // 手机号（唯一）
  password: string,                 // 加密后的密码
  nickname: string,                 // 昵称
  membership_type: string,          // 会员类型: free/monthly/quarterly/yearly/lifetime
  membership_expire_at?: Date,      // 会员过期时间
  today_analyze_count: number,      // 今日分析次数
  total_analyze_count: number,      // 总分析次数
  created_at: Date,                 // 创建时间
  updated_at: Date                  // 更新时间
}
```

**索引**:
- `phone`: 唯一索引
- `membership_type`: 普通索引
- `membership_expire_at`: 普通索引

### 2. lottery_results 集合

开奖结果表

```typescript
{
  _id: ObjectId,
  lottery_type: string,             // 彩票类型: kuaile8
  draw_no: string,                  // 期号（唯一）
  draw_date: Date,                  // 开奖时间
  numbers: number[],                // 开奖号码（15个）
  sum: number,                      // 号码总和
  created_at: Date                  // 创建时间
}
```

**索引**:
- `lottery_type + draw_no`: 唯一复合索引
- `draw_date`: 降序索引
- `lottery_type + draw_date`: 复合索引

### 3. membership_orders 集合

会员订单表

```typescript
{
  _id: ObjectId,
  user_id: ObjectId,                // 用户ID
  order_no: string,                 // 订单号（唯一）
  product_type: string,             // 产品类型: monthly/quarterly/yearly/lifetime
  amount: number,                   // 金额
  payment_method: string,           // 支付方式: wechat/alipay
  status: string,                   // 状态: pending/paid/cancelled
  paid_at?: Date,                   // 支付时间
  created_at: Date                  // 创建时间
}
```

**索引**:
- `order_no`: 唯一索引
- `user_id`: 普通索引
- `status`: 普通索引
- `created_at`: 降序索引

### 4. membership_logs 集合

会员操作日志表

```typescript
{
  _id: ObjectId,
  user_id: ObjectId,                // 用户ID
  action: string,                   // 操作类型: analyze/export/predict
  feature_name: string,             // 功能名称
  result: string,                   // 结果: allowed/denied
  created_at: Date                  // 创建时间
}
```

**索引**:
- `user_id`: 普通索引
- `created_at`: 降序索引

## 中间件

### 1. 认证中间件 (auth.ts)

```typescript
// 使用方式
export async function GET(req: NextRequest) {
  return withAuth(req, async (user) => {
    // user 包含 JWT payload
    // 处理业务逻辑
    return NextResponse.json({ data: 'success' })
  })
}
```

### 2. 会员验证中间件 (membership.ts)

```typescript
// 使用方式
export async function POST(req: NextRequest) {
  return withAuth(req, async (user) => {
    const membershipCheck = await requireMembership(req, 'premium')
    if (membershipCheck instanceof NextResponse) {
      return membershipCheck // 返回错误响应
    }
    
    // 会员验证通过，处理业务逻辑
    return NextResponse.json({ data: 'success' })
  })
}
```

## 安全考虑

### 1. 密码加密

使用 bcryptjs 加密密码：

```typescript
import bcrypt from 'bcryptjs'

// 加密
const hashedPassword = await bcrypt.hash(password, 10)

// 验证
const isValid = await bcrypt.compare(password, hashedPassword)
```

### 2. JWT 认证

```typescript
// 生成 Token
const token = signToken({
  id: user._id.toString(),
  phone: user.phone,
  membershipType: user.membership_type
})

// 验证 Token
const payload = verifyToken(token)
```

### 3. API 限流

建议使用 Redis 实现 API 限流：

```typescript
// 示例：每分钟最多10次请求
const key = `rate_limit:${userId}:${Date.now() / 60000}`
const count = await redis.incr(key)
if (count > 10) {
  return NextResponse.json({ error: '请求过于频繁' }, { status: 429 })
}
await redis.expire(key, 60)
```

### 4. CORS 配置

在 `next.config.js` 中配置：

```javascript
module.exports = {
  async headers() {
    return [
      {
        source: '/api/:path*',
        headers: [
          { key: 'Access-Control-Allow-Origin', value: 'https://your-domain.com' },
          { key: 'Access-Control-Allow-Methods', value: 'GET,POST,PUT,DELETE,OPTIONS' },
          { key: 'Access-Control-Allow-Headers', value: 'Content-Type, Authorization' }
        ]
      }
    ]
  }
}
```

## 常见问题

### Q: 如何重置用户每日分析次数？

A: 创建定时任务，每天凌晨重置：

```typescript
// 每天 00:00 执行
await db.collection('users').updateMany(
  {},
  { $set: { today_analyze_count: 0 } }
)
```

### Q: 如何处理会员过期？

A: 在会员验证中间件中检查过期时间，过期后自动降级为免费用户。

### Q: 如何备份数据库？

A: 使用 MongoDB Atlas 自动备份，或使用 mongodump：

```bash
mongodump --uri="mongodb+srv://..." --out=/backup/
```
