# 快乐8爬虫 - 5分钟快速入门

## 🎯 目标

让你在5分钟内完成数据爬虫的配置和运行。

---

## 📋 准备工作

你需要：
1. ✅ Node.js 已安装
2. ✅ 已进入 `backend-cloudflare/crawler` 目录
3. ✅ 已运行 `npm install`

---

## 🚀 三步搞定

### 第一步：测试API（30秒）

```bash
npm run test:api
```

这会告诉你需要配置哪些API KEY。

### 第二步：注册并获取API KEY（3分钟）

**推荐：数字星空API（免费）**

1. 访问 https://szxk365.com
2. 注册账号
3. 获取API KEY
4. 配置环境变量：

```bash
# Windows CMD
set SZXK_API_KEY=你的API_KEY

# Windows PowerShell
$env:SZXK_API_KEY="你的API_KEY"

# Linux/Mac
export SZXK_API_KEY=你的API_KEY
```

### 第三步：测试运行（30秒）

```bash
# 测试API是否可用
npm run test:api

# 爬取最新数据
npm run crawl:szxk

# 爬取历史数据（可选）
npm run crawl:szxk:history
```

## 🆘 遇到问题？

### 问题1: API KEY无效

**解决：**
- 检查是否正确复制了API KEY
- 确认环境变量是否设置成功
- 重新运行 `npm run test:api` 验证

### 问题2: 网络连接失败

**解决：**
- 检查网络连接
- 尝试访问 https://szxk365.com 确认网站可访问
- 如果在国内，可能需要等待或更换网络

### 问题3: 数据库保存失败

**解决：**
```bash
# 检查wrangler是否登录
cd ..
wrangler whoami

# 如果未登录
wrangler login

# 检查数据库
wrangler d1 list
```

## 💡 小贴士

### 永久配置环境变量

**Windows:**
1. 右键"此电脑" → 属性 → 高级系统设置
2. 环境变量 → 新建
3. 变量名：`SZXK_API_KEY`
4. 变量值：你的API KEY

**Linux/Mac:**
```bash
# 添加到 ~/.bashrc 或 ~/.zshrc
echo 'export SZXK_API_KEY=你的API_KEY' >> ~/.bashrc
source ~/.bashrc
```

### 多个API备份

可以同时配置多个API作为备份：

```bash
export SZXK_API_KEY=你的数字星空KEY
export JISU_APP_KEY=你的极速数据KEY
export APIHZ_ID=你的接口盒子ID
export APIHZ_KEY=你的接口盒子KEY
```

然后通过环境变量切换：

```bash
# 使用数字星空
export CRAWLER_SCRIPT=szxk-crawler.js

# 使用极速数据
export CRAWLER_SCRIPT=jisu-crawler.js
```
