# H5 实现文档

## 项目概述

快乐8百宝箱 H5 版本，基于 React + TypeScript + Vite 构建的移动端 Web 应用。

## 技术栈

- **框架**: React 19
- **语言**: TypeScript
- **构建工具**: Vite 8
- **路由**: React Router DOM 7
- **状态管理**: Zustand 5
- **HTTP 客户端**: Axios 1.7

## 页面说明

### 1. 首页 (Home)

**路由**: `/`

**功能**:
- 展示专区列表
- 快乐8专区入口
- 预留其他专区位置

**特点**:
- 渐变背景（三色）
- 淡入动画效果
- 卡片式布局
- 左侧彩色边条

### 2. 快乐8工具页 (Kuaile8)

**路由**: `/kuaile8`

**功能**:
- 免费工具区：最新开奖、历史数据、基础统计
- 会员工具区：高级分析、AI预测、走势图表、数据导出
- 返回按钮

**交互**:
- 免费工具直接跳转
- 会员工具弹窗提示开通会员

### 3. 最新开奖 (Latest)

**路由**: `/latest`

**功能**:
- 显示最新一期开奖结果
- 期号、开奖时间
- 开奖号码（15个）
- 总和统计

**数据结构**:
```typescript
interface DrawResult {
  issue: string        // 期号
  drawTime: string     // 开奖时间
  numbers: number[]    // 开奖号码
  sum: number          // 总和
}
```

### 4. 历史数据 (History)

**路由**: `/history`

**功能**:
- 显示近期开奖记录
- 列表展示
- 每条记录包含期号、时间、号码、总和

### 5. 基础统计 (BasicStats)

**路由**: `/basic-stats`

**功能**:
- 号码出现频率统计
- 排行榜展示（Top 20）
- 可视化进度条
- 出现次数统计

**数据结构**:
```typescript
interface NumberStat {
  number: number       // 号码
  count: number        // 出现次数
  percentage: number   // 百分比
}
```

### 6. 会员开通 (Membership)

**路由**: `/membership`

**功能**:
- 会员特权展示
- 套餐选择（月度/季度/年度/永久）
- 价格展示
- 立即开通按钮

**套餐**:
- 月度会员：¥29.9 / 30天
- 季度会员：¥79.9 / 90天（推荐）
- 年度会员：¥299 / 365天
- 永久会员：¥999 / 终身

### 7. 个人中心 (Profile)

**路由**: `/profile`

**功能**:
- 用户信息展示
- 会员状态
- 开通会员入口

## API 集成

### 配置代理

在 `vite.config.ts` 中配置：

```typescript
server: {
  proxy: {
    '/api': {
      target: 'http://localhost:3001',
      changeOrigin: true
    }
  }
}
```

### API 调用示例

```typescript
import axios from 'axios'

// 获取最新开奖
const getLatest = async () => {
  const res = await axios.get('/api/kuaile8/latest')
  return res.data
}

// 获取历史数据
const getHistory = async (days: number) => {
  const res = await axios.get(`/api/kuaile8/history?days=${days}`)
  return res.data
}

// 基础统计
const getBasicStats = async () => {
  const res = await axios.get('/api/kuaile8/basic-stats')
  return res.data
}
```

## 部署方案

### 1. Vercel 部署（推荐）

```bash

vercel login

# 部署
vercel
```

**优势**:
- 免费
- 自动 HTTPS
- CDN 加速
- 自动部署

### 2. Nginx 部署

```bash
# 构建
npm run build

# 上传 dist 目录到服务器
scp -r dist/* user@server:/var/www/html/
```

**Nginx 配置**:

```nginx
server {
    listen 80;
    server_name your-domain.com;
    root /var/www/html;
    index index.html;

    # SPA 路由支持
    location / {
        try_files $uri $uri/ /index.html;
    }

    # API 代理
    location /api {
        proxy_pass http://localhost:3001;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }

    # Gzip 压缩
    gzip on;
    gzip_types text/plain text/css application/json application/javascript;
}
```

## 常见问题

### Q: 如何修改后端 API 地址？

A: 修改 `vite.config.ts` 中的 proxy 配置：

```typescript
proxy: {
  '/api': {
    target: 'https://your-api-domain.com',
    changeOrigin: true
  }
}
```

### Q: 如何添加新页面？

A: 
1. 在 `src/pages/` 创建新组件
2. 在 `App.tsx` 添加路由
3. 添加对应的样式文件

### Q: 如何部署到微信公众号？

A: 
1. 构建生产版本
2. 上传到服务器
3. 在公众号后台配置 JS 安全域名
4. 配置网页授权域名
